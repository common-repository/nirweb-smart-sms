افزونه پیامک ووکامرس و ورودپرس | با این افزونه میتوانید انواع اطلاع رسانی های پیامکی برای ووکامرس و وردپرس خود داشته باشید.

== Description ==
پلاگین پیامک پیشرفته نیر وب یک افزونه  کاملا کاربردی است که در نسخه رایگان تمام اطلاع رسانی های مورد نیاز شما در ووکامرس را فراهم میکند.

==افزونه پیامک ووکامرس و وردپرس | رایگان==
* ارسال سریع و بلک لیست در تمامی ارسال ها با استفاده از پترن های دلخواه شما 
* ارسال پیامک به مشتری و مدیرکل و فروشنده در خرید محصول 
* ارسال پیامک به مشتری و مدیرکل و فروشنده در تعییر وضعیت سفارش ها 
* ارسال پیامک برای مدیریت موجودی ( در صورت کم شدن موجودی یا اتمام موجودی محصول به مدیر پیامک هشدار ارسال شود )
* امکان ثبت شماره توسط مشتری برای اطلاع از موجود شدن مجدد محصول
* امکان ثبت شماره توسط مشتری برای اطلاع رسانی فروش ویژه یک محصول
* امکان ویژه ارسال پیامک به مشتری در اولین خرید ( مثلا برای مشتری که اولین خرید خود را از سایت شما دارد کد تخفیف پیامک کنید)

==افزونه پیامک فرم تماس 7 | contact form 7 | رایگان==
* ارسال پیامک هنگام ارسال یک فرم تماس 
* امکان ثبت شماره های متفاوت برای فرم های تماس متفاوت 

==افزونه پیامک لرن دش | رایگان==
* ارسال پیامک به مدیر برای اتمام یک دوره توسط کاربر
* ارسال پیامک به کاربر برای اتمام یک دوره

==برخی از امکانات نسخه تجاری==
* ارسال پیامک برای ثبت نام کاربر جدید ( به مدیر و کاربر )
* امکان ویژه ارسال پیام زمانبندی شده ( میتوانید به یک یا چند کاربر در یک زمان خاص پیام خاصی را ارسال نمایید )
* ارسال پیامک برای پست جدید
* ارسال پیامک به مدیر و کاربر برای دیدگاه و پاسخ دیدگاه
* خبرنامه تخصصی پیامکی 
* افزودن فیلد شماره تماس به ثبت نام وردپرس
* هماهنگی کامل با گراویتی فرم برای ارسال پیامک
* هماهنگی با لرن پرس 
* و ....

===Installation===
1. Upload `nirweb-smart-sms-free` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Complete NirWeb SMART SMS free Settings
